Smart contracts

SzaboDice is a collection of smart contracts that exist in the Ethereum blockchain. Each smart contract operates a decentralized dice casino.

True randomness

SzaboDice gets its randomness from Random.org, which provides true randomness from atmospheric noise, via Oraclize.

Play or invest

Each SzaboDice contract is bankrolled by up to five investors. The highest bidders control the bankroll, and the losers surrender their shares.

Open source

SzaboDice is open source on GitHub. Find the smart contract source code and read it for yourself.

SzaboDice contracts

How to play

Just send Ether to one of the contract addresses listed above. If you win, you'll receive the payout. If you lose, we'll send you 1 Wei so you know you lost. The gas cost will be ~110,000 (0.005 eth). If you're using Geth, here's how to send the transaction:

eth.sendTransaction({from: '0xYOUR_ADDRESS', to: '0xSZABODICE_ADDRESS_HERE', value: web3.toWei(1, "ether"), gas: 110000})

How to invest

Head on over to the GitHub repository to learn how to send investment funds to the smart contract.

SzaboDice has no owner. It is a decentralized autonomous organization governed by the Ethereum blockchain.
